## Generate go files

Firstly check is lib exist `protoc-gen-go` with command

```shell
protoc --version
```

If lib `protoc-gen-go` not exist, download this lib with command 

```shell
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
```
